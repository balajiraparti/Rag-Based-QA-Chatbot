
# RAG Based Question-Answer Chatbot
 A Retrieval Augmented Generation based chatbot where we can ask question about PDF files

## Features
- Developed RAG pipeline that efficiently retrives relevant chunks/docs from vector database
- Added multple PDF support
- Integrated RRF(Reciprocal Rank Fusion) technique to handle vague user query by generating parallel queries and re-ranking docs based on there occurences



## Tech Stack

- **Python**
- **LangChain**
- **OpenAI API**
- **Qdrant Vector Database**
- **Streamlit**
